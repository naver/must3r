# Updated Evaluation results

> [!NOTE]  
> In this released, we fixed a bug in the slam where some frames were in some rare cases, valid keyframes were incorrectly rejected. It improves the results slighly on these edge cases.

```
Here's an example:
MUSt3R_224_cvpr-C on TUM RGBD (--res 224 --subsamp 2 --keyframe_overlap_thr 0.05 --min_conf_keyframe 1.5 --overlap_percentile 85)

before the fix
Mean ALL RMSE= 25.9 cm
Median ALL RMSE= 7.6 cm
Avg scaling error: 31.86896%
Median scaling error: 17.90361%

after the fix
Mean ALL RMSE= 24.3 cm
Median ALL RMSE= 7.6 cm
Avg scaling error: 31.08055%
Median scaling error: 16.98365%

```
